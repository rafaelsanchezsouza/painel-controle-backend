Hello World!

This is the back-end project developed for the [front-end dashboard](https://github.com/rafaelsanchezsouza/painel-controle-frontend).

### Built With

- [TypeScript](https://www.typescriptlang.org/)
- [TypeORM](https://typeorm.io/#/)
- [Express](https://expressjs.com/pt-br/)

## Getting Started

### Requirements

- NPM / Yarn

```
npm install -g yarn
```

### Instalation

1. Clone Repository

```
git clone https://github.com/rafaelsanchezsouza/painel-controle-backend.git
```

2. Install required packages

```
yarn install
```

3. Start local application

```
yarn dev
```

## How to use this API

### Routes

POST /empresas
POST /unidades
POST /usuarios
POST /ativos
POST /status

GET /empresas

GET /usuarios
GET /ativos
